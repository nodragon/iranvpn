# Legal disclaimer and operational security (US-008)

## Legal notice

Unauthorized VPN use is **illegal** in Iran (Supreme Council of Cyberspace, February 2024).

This project and any resulting software are for **research and informational use only**. We do not encourage illegal activity. Users assume all legal and personal risks. The maintainers are not responsible for how the software is used.

## Operational security recommendations

- Use **DNS-over-HTTPS (DoH)** or tunneled DNS when possible to reduce poisoning and leakage.
- Avoid public or untrusted Wi-Fi for sensitive use.
- Keep the application and operating system **updated**.
- Install the app **before** anticipated shutdowns when possible (US-005).
